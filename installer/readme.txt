====================================================
YOUTUBE PLAYLIST DOWNLOADER - INSTALLATION GUIDE
====================================================

Thank you for downloading YouTube Playlist Downloader, a simple application
that lets you download YouTube videos as audio files and keep your collection
up to date automatically.

QUICK INSTALLATION:
------------------

1. Double-click on 'YouTubePlaylistDownloader_Setup.exe'
2. Follow the on-screen prompts
3. After installation completes, start the application using the desktop shortcut

SYSTEM REQUIREMENTS:
------------------

- Windows 7/8/10/11
- 100 MB of free disk space (plus space for your downloads)
- Internet connection

WHAT THIS APPLICATION DOES:
-------------------------

- Downloads YouTube videos as audio files (MP3, WAV, etc.)
- Tracks playlists to automatically download new videos
- Maintains a history of downloaded files to avoid duplicates
- Converts audio to different formats if needed

GETTING STARTED:
--------------

After installation:

1. Launch the application from your desktop shortcut
2. Go to the 'Settings' tab to configure your preferred output directory
3. Add playlists to track in the 'Playlists' tab
4. Use 'Download Video' tab for single video downloads

For detailed instructions, please refer to the User Guide included in this package.

TROUBLESHOOTING:
--------------

If you encounter any issues:

- Check if an error_log.txt file was created in the application folder
- Make sure you have a stable internet connection
- Verify that the YouTube URLs you're using are correct
- Try restarting the application

UNINSTALLING:
-----------

To uninstall the application:

1. Go to Control Panel > Programs > Uninstall a program
2. Select "YouTube Playlist Downloader" from the list
3. Click "Uninstall" and follow the prompts

LEGAL NOTICE:
-----------

This application is for personal use only. Please respect copyright laws and
only download content you have permission to download and use according to
YouTube's terms of service.
